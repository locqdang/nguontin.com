# Contracts

This directory is reserved for explicit interface contracts created during later Spec Kit planning passes.

Add files here only when the feature needs concrete artifacts such as:

- API endpoint contracts
- request or response schemas
- webhook payload contracts
- integration boundary notes

For the current NguonTin MVP planning baseline, the canonical sources remain:

- `spec.md`
- `plan.md`
- `data-model.md`
- `tasks.md`
